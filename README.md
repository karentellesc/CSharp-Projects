# Simulador de Caixa Eletrônico
Esse foi um projeto realizado em C# para criação de um simulador de caixa eletrônico.

Nesse projeto, o usuário irá realizar um saque. Primeiro passo é aparecer quais as notas estão disponíveis naquele caixa e logo depois irá solicitar que o usuário digite o valor.

Irá aparecer uma mensagem de confirmação se é realmente aquele valor e o usuário terá a opção de prosseguir o saque ou digitar um novo valor.

Se tiver notas disponíveis, irá aparecer a mensagem de "Saque realizado com sucesso". Caso não haja notas disponíveis, irá aparecer a mensagem comunicando que não será possível a realização do saque e solicitando um novo valor.
